# 알고리즘 학습 일지 (BOJ)

##  날짜: 2025년 5월 14일
## 문제: 알람 시계 (BOJ #2884)
### 문제 링크
https://www.acmicpc.net/problem/2884
### 문제 요약
입력된 시와 분에서 45분을 빼서 새로운 시각을 계산하는 문제.
### 입력
10 10

### 출력
9 25

### 풀이 코드 (Java)
h, m = map(int, input().split())

m -= 45
if m < 0:
    m += 60
    h -= 1
    if h < 0:
        h = 23

print(h, m)

###  학습 내용
조건문 안에서 수학 계산을 함께 처리하며 시간 경계 조건에 주의하는 법 학습.

### 느낀 점
실제 생활 문제를 코드로 구현하는 과정이 재밌었고, 조건 순서의 중요성도 다시 체감함.
